# PragyanAI

A responsive study assistant website built with plain HTML, CSS, and JavaScript.

## Run locally

Open the project folder in a browser or use a simple server:

```bash
python -m http.server 8000
```

Then visit:

```text
http://127.0.0.1:8000/
```

## Publish online

The simplest way is to upload these files to GitHub Pages or any static hosting service such as Netlify, Vercel, or Cloudflare Pages.

### GitHub Pages
1. Create a repository on GitHub.
2. Upload the contents of this folder.
3. In repository settings, enable Pages.
4. Choose the main branch and root folder.
5. Your site will be published at:
   `https://<your-username>.github.io/<repository-name>/`
